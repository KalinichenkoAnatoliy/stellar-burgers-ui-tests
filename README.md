# 🍔 Stellar Burgers: UI Autotests

Автотесты пользовательского интерфейса (UI) для сервиса доставки бургеров **Stellar Burgers**.  
Проект выполнен в рамках дипломной работы и демонстрирует навыки веб-автотестирования с использованием **Selenium WebDriver**, паттерна **Page Object** и формирования отчётов в **Allure**.

## 🎯 Задача
1. Описать элементы интерфейса с помощью паттерна **Page Object**.
2. Протестировать основную функциональность приложения.
3. Запустить тесты в браузерах **Google Chrome** и **Mozilla Firefox**.
4. Подключить отчётность **Allure**.

## 🛠 Стек технологий
- **Язык:** Python 3.x
- **Фреймворк:** pytest
- **WebDriver:** Selenium
- **Отчетность:** allure-pytest
- **Паттерн:** Page Object Model (POM)
- **Браузеры:** Chrome, Firefox
